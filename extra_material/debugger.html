How do you solve a problem?

Use the debugger!

Instructions:

  1. Open this file in Chrome
  2. Open developer tools
  3. In the JavaScript console, inspect some_object by typing the variable name in the console.
  4. You can run any JavaScript you like in the console as if you were at that line of code in the execution of your program.

You can access properties on an object just like you would in your JS code!

  • some_object.name
  • some_object.age

Notice how much time this can save you to diagnose issues. If a function is broken, put a debugger where you think the issue is and JavaScript will enter the console from that spot.
